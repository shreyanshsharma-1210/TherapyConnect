# TherapyConnect UI Components

Complete list of all UI components used in the TherapyConnect application.

## UI Components (`src/components/ui/`)

- **Spinner** — Loading spinner with size variants
- **Skeleton** — Loading skeleton with SkeletonCalendar and SkeletonTimeSlots variants
- **Modal** — Modal dialog with backdrop
- **Input** — Form input field with error states
- **Card** — Card container with variants
- **Button** — Button with variants (primary, secondary, ghost, danger)
- **Badge** — Badge component for status/tags

## Layout Components (`src/components/layout/`)

- **Layout** — Main layout wrapper with navbar and footer
- **Navbar** — Navigation header with mobile menu
- **Footer** — Page footer with links and social icons

## Common Components (`src/components/common/`)

- **Container** — Responsive container wrapper
- **Section** — Section wrapper with padding
- **Heading** — Heading component with variants
- **SEO** — SEO meta tags and structured data
- **RouteTransition** — Page transition animation wrapper
- **ScrollToTop** — Scroll to top on route change
- **FloatingWhatsApp** — Floating WhatsApp chat button
- **StickyMobileCTA** — Sticky call-to-action for mobile
- **CookieConsent** — Cookie consent banner

## Auth Components (`src/components/auth/`)

- **AuthLayout** — Authentication page layout wrapper
- **AuthField** — Form field with icon and error state
- **ProtectedRoute** — Route protection for authenticated users
- **AdminRoute** — Route protection for admin users only
- **GuestRoute** — Route protection for unauthenticated users

## Booking Components (`src/components/booking/`)

- **BookingCalendar** — Calendar date picker for booking
- **TimeSlotPicker** — Time slot selection grid
- **BookingForm** — Multi-step booking form
- **BookingSummary** — Booking details summary
- **BookingConfirmation** — Booking confirmation with payment
- **BookingStepIndicator** — Step progress indicator

## Payment Components (`src/components/payment/`)

- **PaymentModal** — Payment modal with Razorpay/demo mode

## Notification Components (`src/components/notifications/`)

- **NotificationCenter** — Notification dropdown panel
- **NotificationBell** — Notification bell icon with badge

## Section Components (`src/components/sections/`)

- **HeroSection** — Landing page hero with CTA
- **AboutSection** — Therapist about section
- **ServicesSection** — Services cards grid
- **PricingSection** — Pricing plans display
- **BookingSection** — Booking CTA section
- **TestimonialsSection** — Testimonials carousel
- **StatsBar** — Statistics/metrics bar
- **BlogSection** — Blog posts preview
- **ContactSection** — Contact form and info
- **CTASection** — Call-to-action banner

## Admin Components (`src/components/admin/`)

- **AdminLayout** — Admin dashboard layout with sidebar
- **PageHeader** — Admin page header with title and actions
- **StatCard** — Statistics card with icon and trend
- **AdminCard** — Admin content card wrapper
- **StatusBadge** — Status badge for bookings
- **AdminBtn** — Admin button with variants
- **AdminInput** — Admin form input
- **AdminTextarea** — Admin textarea field
- **AdminSelect** — Admin select dropdown
- **EmptyState** — Empty state placeholder
- **ConfirmDialog** — Confirmation dialog modal
